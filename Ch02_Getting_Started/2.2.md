### 2.2-2

```
SELECT(A)
for i = 1 to A.length - 1
    min = i
    for j = i + 1 to A.length
        if A[j] < A[min]
            min = j
    swap(A[min], A[i])
```

循环不变式是数组的前 i-1 个元素 A[1...i-1] 始终有序

因为前 n-1 个元素排序完成后，第 n 个元素自然是最大的

都是θ(n^2)
